# Hostinger + mobile OTP

1. Deploy `main`.
2. Set env vars (no quotes):

```env
SUPABASE_URL=...
SUPABASE_SERVICE_ROLE_KEY=...
SUPABASE_ANON_KEY=...
AUTH_SECRET=long-random-secret
AUTH_URL=https://ilovebiodata.com
OTP_DEV_MODE=false
MSG91_AUTH_KEY=...
MSG91_TEMPLATE_ID=...
RAZORPAY_KEY_ID=...
RAZORPAY_KEY_SECRET=...
NEXT_PUBLIC_RAZORPAY_KEY_ID=...
```

3. Run SQL migrations `001`, `002`, and `003` in Supabase.
4. Rebuild/restart Node.js.
5. Verify `/api/health/database`.
6. Test login with a real phone number.
